## Quickstart

__Config__

Add the following to your `.env` and edit accordingly

- Validation Key is used for Jarvis setup
- SUPERDAMIN_EMAIL for DashAuth

``` env
VALIDATION_KEY=1234567890_mykey

SUPER_ADMIN_EMAIL=shawnsandy04@gmail.com
```
